## 1. Introduction

Singular Value Decomposition (SVD) is one of the most fundamental and powerful matrix decomposition techniques in linear algebra. It acts as the "swiss-army knife" of data science, serving as the underlying engine for Principal Component Analysis (PCA), recommendation systems, image compression, and noise reduction.

At its core, SVD is the process of factorizing any $m×n$ matrix $A$ into the product of three special matrices:
$A=U\Sigma V^{T}$



-    Where: $U$ is an $m×n$ **unitary matrix** (Specifically, its columns are orthonormal vectors. This means $U^{TU}=I$ , the identity matrix).
-   Σ (Sigma) is an $m×n$ **rectangular diagonal matrix**. All elements are zero except the diagonal entries, which are non-negative real numbers called **singular values**.
-   $V^T$ is an $n×n$ **unitary matrix** (the transpose of $V$, whose columns are also orthonormal vectors).

### Terminology

-   The diagonal entries of $Σ$ are the **Singular Values** of $A$
    
    .
-   The columns of $U$ are the **Left-Singular Vectors** of $A$
    
    .
-   The columns of $V$ are the **Right-Singular Vectors** of $A$
    
    .

----------

## 2. SVD vs. Eigendecomposition

If you are familiar with Eigendecomposition ($A=QΛQ^{−1}$), SVD will look very similar. However, SVD is strictly more general and has two critical advantages:
| Feature | Eigendecomposition | Singular Value Decomposition (SVD) |
| --- | --- | --- |
| Matrix Requirements | Must be a square (n×n) matrix. | Works on any matrix (m×n). |
| Eigenvalue Ordering | Eigenvalues can be in any order. | Singular values are always sorted in descending order (σ1​≥σ2​≥…). |

The descending order of singular values is incredibly useful. It means the first column of $U$ and the first row of $V^T$ capture the most variance/information in the matrix, the second captures the second most, and so on.
## 3. Basic Mathematical Example

Let's decompose a $3×2$ matrix $A$





$$
A = \begin{bmatrix}
1 & -1 \\
0 & 1 \\
1 & 0
\end{bmatrix}
$$


Using SVD, this is decomposed as:



$$
\begin{bmatrix}
1 & -1 \\
0 & 1 \\
0 & 0
\end{bmatrix} = 
\begin{bmatrix}
-\frac{2}{\sqrt{6}} & 0 & -\frac{1}{\sqrt{3}} \\
\frac{1}{\sqrt{6}} & -\frac{1}{\sqrt{2}} & \frac{1}{\sqrt{3}} \\
0 & 0 & 0
\end{bmatrix}
\begin{bmatrix}
\sqrt{3} & 0 \\
0 & 1 \\
0 & 0
\end{bmatrix}
\begin{bmatrix}
-\frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{2}} \\
-\frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{2}}
\end{bmatrix}
$$




### Verifying the Properties

1.  **Unit Vectors:** Look at the first column of U: $\left[-2/\sqrt{6}\quad1/\sqrt{6}\quad-1/\sqrt{6}\right]^{T}$
    
2. Its magnitude is  $\sqrt{(-2/\sqrt{6})^{2}+(1/\sqrt{6})^{2}+(-1/\sqrt{6})^{2}}=\sqrt{4/6+1/6+1/6}=1$ . It is a unit vector.
3.  **Orthogonality:** The dot product of the 2nd and 3rd columns of $U$ is: $0\left(-\frac{1}{\sqrt{3}}\right) + \left(-\frac{1}{\sqrt{2}}\right)\left(-\frac{1}{\sqrt{3}}\right) + \left(-\frac{1}{\sqrt{2}}\right)\left(\frac{1}{\sqrt{3}}\right) = 0 + \frac{1}{\sqrt{6}} - \frac{1}{\sqrt{6}} = 0$ . They are orthogonal.
8.  **Descending Order in** $Σ$: The diagonal values are ${\sqrt{3}}\approx1.73$ and 1. They are in descending order.


## 4. Python Implementation: Basic SVD

When performing SVD in Python using `scipy.linalg.svd` or `numpy.linalg.svd`, the function optimizes for memory. Instead of returning a full $m×n$ matrix filled with zeros for $Σ$, it returns a **1D array** containing only the singular values.


## Script

```python

import numpy as np
from scipy import linalg

# Format output for readability
np.set_printoptions(precision=3, suppress=True)  # Suppress scientific notation for small numbers and round to 3 decimal places for better readability.

# Define a 3x2 matrix (m=3, n=2). 
A = np.array([[1, -1],
            [0,  1],
            [1,  0]])

# Perform SVD. The function linalg.svd returns three matrices: U, S, and Vt.
# U is an m x m orthogonal matrix (3x3 in this case).
# S is a 1D array of singular values (length min(m, n) = 2 in this case).
# Vt is the transpose of V, where V is an n x n orthogonal matrix (2x2 in this case).
U, S, Vt = linalg.svd(A)

print("Matrix U Shape:", U.shape)  # Output: (3, 3)
print("Matrix U:\n", U)  # Output: A 3x3 matrix where columns are the left singular vectors of A.
print("\n1D Array S Shape:", S.shape)  # Output: (2,) because S contains only the non-zero singular values, which are 2 in this case.
print("Array S:", S)  # Output: [1.618, 1.000] (the singular values of A, sorted in descending order)
print("\nMatrix Vt Shape:", Vt.shape)  # Output: (2, 2) because Vt is the transpose of V, which is a 2x2 matrix in this case.
print("Matrix Vt:\n", Vt)  # Output: A 2x2 matrix where rows are the right singular vectors of A (or columns of V, since Vt is the transpose of V).

# --- RECONSTRUCTION ---
# To multiply U @ S @ Vt, we must convert the 1D array S into a 3x2 matrix.
# We create a matrix of zeros with the same shape as A, and fill the diagonal.
Sigma = np.zeros(A.shape)  # Create a matrix of zeros with the same shape as A (3x2).
np.fill_diagonal(Sigma, S)  # Fill the diagonal of Sigma with the singular values from S.

# Reconstruct A. The original matrix A can be reconstructed by multiplying U, Sigma, and Vt together.
A_reconstructed = U @ Sigma @ Vt  # The @ operator is used for matrix multiplication in Python. It multiplies U by Sigma, and then the result by Vt to reconstruct A.

print("\nReconstructed Matrix:\n", A_reconstructed)  # Output: A matrix that should closely match the original matrix A, confirming that the SVD decomposition and reconstruction are correct.
# Output matches A exactly (up to floating point precision)


```
