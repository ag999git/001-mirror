**Q20. Write a script that uses a match-case statement to check the tuple variable status = (404, "Not Found") and print a custom message based on its contents.**

```python
    case (200, message):
        print(f"Success Content: {message}")
    case (404, message):
        # Step 3: Print the message when the exact pattern triggers
        print(f"Handled Error Message: {message}")
```
HOW PYTHON WORKS EXPLANATION:
•	Line `match status:` and `case (404, message):`- In modern Python, the match-case statement inspects both the structure and values of a tuple simultaneously.
•	The engine verifies that the data is a two-element tuple, checks if the first item matches the literal number `404`, and then automatically extracts the second item, binding it instantly to the local variable message.
